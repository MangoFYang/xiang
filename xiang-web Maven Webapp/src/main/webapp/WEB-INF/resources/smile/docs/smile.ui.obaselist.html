obaselist

l  方法

方法名

参数:类型

描述

getData

node:Object

获取某节点的原始数据(组件不会丢失用户提供的任何数据)。node：获取原始数据的对象。获取当前选中节点可使用getSelected方法。

setData

data:Array

设置数据集。数据集结构为(数组中是对象):

[{

    key: value

}, {

    key: value

}, …]

getSelected

 

获取选中节点。选中多个返回节点的数组。

refresh

 

初始化数据。当动态添加数据项时需要调用。

select

value:Mixed

选中节点。Mixed表示：Object|String|HTML Element等类型的参数都支持。

unselect

value:Mixed

取消选中节点。Mixed表示：Object|String|HTML Element等类型的参数都支持。

selectAll

 

选中所有节点, 只读multiSelect为true是生效。

unselectAll

 

取消所有选中节点。
