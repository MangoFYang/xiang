odialog

l  描述及最佳实践

一个对话框是一个浮动窗口，其中包含一个标题栏和内容区域。对话窗口可以移动，调整大小和关闭的'X'图标。默认情况下，如果内容长度超过了最大高度，滚动条会自动出现。

 

 

 

继承：$.smile.obase
