ocombobox

l  描述及最佳实践

具有自动搜索的单选框，如果将readOnlyInput选项设置为true，则变成和HTML中select功能相似的组件。

是否设置readOnlyInput为true需要使用者根据数据多少而定。

 

 

继承：$.smile.olist4input
