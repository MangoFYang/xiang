l  成员方法getValue和静态方法getValue的区别。

n  定义

成员方法：每一个ocheckbox或oradio组件都有getValue方法，此方法称为ocheckbox或oradio的成员方法。

静态方法：在命名空间$.smile.ocheckbox和$.smile.oradio下都有getValue方法，此方法成为静态方法。

n  用途

成员方法getValue获取的值为此组件的value值，而静态方法获取ocheckbox或oradio组(多个name属性相同的ocheckbox或oradio被确定为一组)被选中的值。
