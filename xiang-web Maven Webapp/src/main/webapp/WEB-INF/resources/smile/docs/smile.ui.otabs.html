otabs

l  描述及最佳实践

otabs通常被用来打破的内容分成多个部分，可以交换，以节省空间，就像一个手风琴。一个otabs组件由多个tab组成，具体可参见item选项。

 

 

 

继承：$.smile.obase

l  方法

方法名

参数:类型

描述

select

index:int

选中一个标签页，就像被点击一样。index：标签页的索引，从零开始。

load

index:int

当标签页的内容需要发请求动态加载时，调用此方法可以加载标签页。

length

 

获取标签的个数。

abort

 

终止所有的AJAX请求和动画。

enable

index:int

启用索引号为index的tab。index：标签页的索引，从零开始。

disable

index:int

禁用索引号为index的tab。index：标签页的索引，从零开始。

getSelected

 

获取当前选中的tab页，返回tab页的下标，从0开始。
